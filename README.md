# 🌍 Wanderlust

A full-stack travel listing web application built using Node.js, Express, MongoDB, and EJS.

---

## 🚀 Features
- User authentication
- Create & manage travel listings
- Image uploads
- MongoDB database integration
- Responsive UI

---

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- EJS
- Bootstrap

---
